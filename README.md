# 使用
- 引用使用
```npm
 npm install element-ui-yh -S
```
```script
  import Vue from 'vue'
  import Element from 'element-ui-yh'

  Vue.use(Element)
```

- 下载源码启动项目后，在localhost:8085中查看具体组件使用，后期会部署到服务器查看
